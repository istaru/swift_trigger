# Triage 标签

各技能使用五个标准 triage 角色。本文件将这些角色映射到此仓库议题追踪器中实际使用的标签字符串。

| 技能中的角色 | 本仓库标签 | 含义 |
| ------------ | ---------- | ---- |
| `needs-triage` | `needs-triage` | 维护者需要评估此议题 |
| `needs-info` | `needs-info` | 等待提问者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整描述，可交由 AFK Agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

当技能提到某个角色（例如「打上 AFK-ready triage 标签」）时，使用上表右列对应的标签字符串。

若要修改标签名，直接编辑右列即可。
